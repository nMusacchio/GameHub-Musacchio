# Gamehub
Proyecto para el curso de ReactJS de Coderhouse.

Se trata de un e-commerce de videojuegos.

## Instalación
```
npm i
npm start
```

## Vistas
### Home
```
localhost:3000/
```
### Catálogo
```
localhost:3000/
localhost:3000/category/:id_category
```
### Item
```
localhost:3000/
localhost:3000/item/:id_item
```
### Cart
```
localhost:3000/
localhost:3000/cart
```
